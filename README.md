# 2021 KAUPC

항공대 알고리즘 학회, KOALA에서 제작한 2021 KAUPC 홍보 페이지입니다.  
`Nuxt.js` 및 `Tailwind CSS` 기반으로 제작되었습니다.

## 👨🏻‍💻 기여 방법

### 프로젝트 클론 및 실행

```
git clone https://github.com/본인-깃허브-닉네임/KAUPC-2021.git
yarn
yarn dev
```
